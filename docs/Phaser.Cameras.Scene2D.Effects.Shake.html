new Shake( camera )

Description
A Camera Shake effect. This effect will shake the camera viewport by a random amount, bounded by the specified intensity, each frame. Only the camera viewport is moved. None of the objects it is displaying are impacted, i.e. their positions do not change. The effect will dispatch several events on the Camera itself and you can also specify an `onUpdate` callback, which is invoked each frame for the duration of the effect if required.

Description
The intensity of the effect. Use small float values. The default when the effect starts is 0.05. This is a Vector2 object, allowing you to control the shake intensity independently across x and y. You can modify this value while the effect is active to create more varied shake effects.
